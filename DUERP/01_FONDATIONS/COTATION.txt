  PRINCIPE GÉNÉRAL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  La cotation se déroule en 2 temps successifs :

  ┌─────────────────────────────────────────────────────────────────────────────────┐
  │                                                                                 │
  │   ÉTAPE 1                        ÉTAPE 2                                       │
  │                                                                                 │
  │   Criticité Brute          ×    Plan de Maîtrise    =   Criticité Résiduelle   │
  │   (G × P)                        (PM)                                          │
  │                                                                                 │
  └─────────────────────────────────────────────────────────────────────────────────┘

  ÉTAPE 1 — CRITICITÉ BRUTE  (G × P)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ── GRAVITÉ (G) ───────────────────────────────────────────────────────────────────

  ┌───────┬──────────────────────────────────────────────────────────────────────┐
  │   G   │  Description                                                         │
  ├───────┼──────────────────────────────────────────────────────────────────────┤
  │   1   │  Accident sans soin                                                  │
  │   2   │  Accident avec soin, sans incapacité                                 │
  │   3   │  Accident avec incapacité partielle (temporaire)                     │
  │   4   │  Accident avec incapacité permanente / Maladie professionnelle       │
  │   5   │  Décès                                                               │
  └───────┴──────────────────────────────────────────────────────────────────────┘

  ── PROBABILITÉ (P) ───────────────────────────────────────────────────────────────

  ┌───────┬──────────────────────────────────────────────────────────────────────┐
  │   P   │  Description                                                         │
  ├───────┼──────────────────────────────────────────────────────────────────────┤
  │   1   │  Improbable — n'est jamais arrivé dans cette situation              │
  │   2   │  Probable — ça peut arriver, on peut l'imaginer                     │
  │   3   │  Certain — arrive parfois, des précédents existent                  │
  │   4   │  Fatal — ça arrive régulièrement / tout le temps                    │
  └───────┴──────────────────────────────────────────────────────────────────────┘

  ── MATRICE DE CRITICITÉ BRUTE  (G × P) ──────────────────────────────────────────

                     PROBABILITÉ
                  P=1    P=2    P=3    P=4
               ┌──────┬──────┬──────┬──────┐
          G=1  │  1   │  2   │  3   │  4   │
               ├──────┼──────┼──────┼──────┤
          G=2  │  2   │  4   │  6   │  8   │
               ├──────┼──────┼──────┼──────┤
          G=3  │  3   │  6   │  9   │  12  │
               ├──────┼──────┼──────┼──────┤
          G=4  │  4   │  8   │  12  │  16  │
               ├──────┼──────┼──────┼──────┤
          G=5  │  5   │  10  │  15  │  20  │
               └──────┴──────┴──────┴──────┘

  ── ZONES DE CRITICITÉ ────────────────────────────────────────────────────────────

  ┌──────────────┬───────────────────┬────────────────────────────────────────────┐
  │  Score       │  Niveau           │  Interprétation                            │
  ├──────────────┼───────────────────┼────────────────────────────────────────────┤
  │   1  à  4    │  VERT  — Faible   │  Risque acceptable, surveillance suffisante│
  │   5  à  9    │  JAUNE — Modéré   │  Actions à planifier                       │
  │  10  à  14   │  ORANGE — Élevé   │  Actions prioritaires à mettre en place    │
  │  15  à  20   │  ROUGE — Critique │  Actions immédiates requises               │
  └──────────────┴───────────────────┴────────────────────────────────────────────┘

  ÉTAPE 2 — PLAN DE MAÎTRISE (PM)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Le plan de maîtrise reflète les moyens de protection déjà en place.
  Plus les protections sont complètes, plus le score final est réduit.

  ┌─────────────────────────────────────────────────────────────┬─────────────────┐
  │  Situation de maîtrise                                      │  Multiplicateur │
  ├─────────────────────────────────────────────────────────────┼─────────────────┤
  │  Aucune protection en place (sans EPI ni mesure)            │     × 1,0       │
  ├─────────────────────────────────────────────────────────────┼─────────────────┤
  │  EPI seuls OU mesures organisationnelles légères            │     × 0,75      │
  │  (protection partielle)                                     │                 │
  ├─────────────────────────────────────────────────────────────┼─────────────────┤
  │  Protections collectives + EPI (double protection)          │     × 0,50      │
  ├─────────────────────────────────────────────────────────────┼─────────────────┤
  │  Protections collectives robustes + procédures formalisées  │     × 0,25      │
  │  + formation                                                │                 │
  ├─────────────────────────────────────────────────────────────┼─────────────────┤
  │  Risque totalement supprimé                                 │     × 0,0       │
  │  (ne doit plus apparaître dans l'EVR)                       │                 │
  └─────────────────────────────────────────────────────────────┴─────────────────┘

  Les protections sont classées selon la hiérarchie T.H.O. + EPI :
    T = Technique       (équipements, protections collectives physiques)
    H = Humain          (binôme, rotation, réduction du temps d'exposition)
    O = Organisationnel (procédures, plannings, signalétique, formation)
    EPI = Équipements de Protection Individuelle (dernier recours)

  FORMULE COMPLÈTE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ┌─────────────────────────────────────────────────────────────────────────────────┐
  │  Criticité Brute       =  Gravité (G)  ×  Probabilité (P)                     │
  │  Criticité Résiduelle  =  Criticité Brute  ×  PM                              │
  │  Criticité Finale      =  Criticité Résiduelle après Plan d'action            │
  └─────────────────────────────────────────────────────────────────────────────────┘

  Exemple concret :
    · Risque : chute de plain-pied sur sol glissant
    · G = 3 (incapacité partielle possible)
    · P = 3 (arrive parfois — précédents connus)
    · Criticité brute = 3 × 3 = 9 → JAUNE
    · PM = 0,75 (un tapis anti-glisse est en place)
    · Criticité résiduelle = 9 × 0,75 = 6,75 → JAUNE
    · Plan d'action prévu : revêtement de sol anti-dérapant permanent
    · PM cible = 0,25
    · Criticité finale = 9 × 0,25 = 2,25 → VERT

  DÉCISION EN ATTENTE — P×G ou P×G×E ?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  La méthode actuelle utilise G × P (2 facteurs).
  Certaines méthodes INRS recommandent d'ajouter un facteur E (Exposition /
  Fréquence d'exposition) pour affiner l'évaluation.

  ┌──────────────────┬──────────────────────────────┬───────────────────────────────┐
  │                  │  G × P  (actuel)             │  G × P × E  (option)          │
  ├──────────────────┼──────────────────────────────┼───────────────────────────────┤
  │  Avantages       │  Simple, rapide              │  Plus précis                  │
  │                  │  Accessible aux non-HSE      │  Reflète la réalité terrain   │
  │                  │  Cohérent avec la source      │  Recommandé par certaines     │
  │                  │  (Cotation.png)               │  méthodes INRS                │
  ├──────────────────┼──────────────────────────────┼───────────────────────────────┤
  │  Inconvénients   │  Moins nuancé                │  Troisième facteur à définir  │
  │                  │  (expo 1h/an = expo 8h/jour) │  Peut complexifier la saisie  │
  └──────────────────┴──────────────────────────────┴───────────────────────────────┘

  Échelle E si retenu :
    1 = Rare (quelques fois par an)
    2 = Occasionnel (quelques fois par mois)
    3 = Fréquent (quelques fois par semaine)
    4 = Permanent (quotidien)
